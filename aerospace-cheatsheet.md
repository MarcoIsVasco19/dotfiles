# AeroSpace Keyboard Shortcuts Cheatsheet

## Layout Management
| Shortcut | Action |
|----------|--------|
| `Alt + /` | Toggle tiles layout (horizontal/vertical) |
| `Alt + ,` | Toggle accordion layout (horizontal/vertical) |

## Focus Navigation (Vim-style)
| Shortcut | Action |
|----------|--------|
| `Alt + H` | Focus left |
| `Alt + J` | Focus down |
| `Alt + K` | Focus up |
| `Alt + L` | Focus right |

## Move Windows
| Shortcut | Action |
|----------|--------|
| `Alt + Shift + H` | Move window left |
| `Alt + Shift + J` | Move window down |
| `Alt + Shift + K` | Move window up |
| `Alt + Shift + L` | Move window right |

## Resize Windows
| Shortcut | Action |
|----------|--------|
| `Alt + -` | Decrease window size by 50px |
| `Alt + =` | Increase window size by 50px |

## Switch to Workspace
| Shortcut | Workspace |
|----------|-----------|
| `Alt + 1-9` | Go to workspace 1-9 |
| `Alt + B` | Browser |
| `Alt + C` | Chats (Slack/ConnectTeam) |
| `Alt + E` | File Explorer (Finder) |
| `Alt + M` | Music |
| `Alt + N` | Notes/Notion |
| `Alt + T` | Terminal |
| `Alt + V` | VSCode |

## Move Window to Workspace
| Shortcut | Action |
|----------|--------|
| `Cmd + Ctrl + Alt + Shift + 1-9` | Move window to workspace 1-9 |
| `Cmd + Ctrl + Alt + Shift + B` | Move to Browser workspace |
| `Cmd + Ctrl + Alt + Shift + C` | Move to Chats workspace |
| `Cmd + Ctrl + Alt + Shift + E` | Move to Explorer workspace |
| `Cmd + Ctrl + Alt + Shift + M` | Move to Music workspace |
| `Cmd + Ctrl + Alt + Shift + N` | Move to Notes workspace |
| `Cmd + Ctrl + Alt + Shift + T` | Move to Terminal workspace |
| `Cmd + Ctrl + Alt + Shift + V` | Move to VSCode workspace |

## Workspace Management
| Shortcut | Action |
|----------|--------|
| `Alt + Tab` | Switch to previous workspace |
| `Alt + Shift + Tab` | Move workspace to next monitor |

## Service Mode
| Shortcut | Action |
|----------|--------|
| `Alt + Shift + ;` | Enter service mode |

### Service Mode Commands (when in service mode)
| Shortcut | Action |
|----------|--------|
| `Esc` | Reload config & exit service mode |
| `R` | Reset layout (flatten workspace tree) & exit |
| `F` | Toggle floating/tiling layout & exit |
| `Backspace` | Close all windows except current & exit |
| `Alt + Shift + H/J/K/L` | Join window with left/down/up/right & exit |
| `↓` | Volume down |
| `↑` | Volume up |
| `Shift + ↓` | Mute (set volume to 0) & exit |
